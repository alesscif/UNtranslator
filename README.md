# UNtranslator
Translates traditional Tekken notation to Universal Numpad notation

Simplified conversion table:

a,b,c,d = 1,2,3,4

1,2,3,4,5,6,7,8,9 = db,d,df,b,n,f,ub,u,uf

[] = DB,D,DF etc. (held direction) and * (held button)

SS+/SS- = SSL/SSR

SW+/SW- = SWL/SWR

236, 214 = qcf, qcb

41236, 63214 = hcf, hcb

623, 421 = dp, rdp 

6523 = CD

WR/iwr = WS/iws

R/ir = WR/iwr

x = cancel 

FCx = cc
